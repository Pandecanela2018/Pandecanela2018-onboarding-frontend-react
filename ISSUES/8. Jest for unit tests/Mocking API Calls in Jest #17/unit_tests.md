Why is it important to mock API calls in tests?
mocking API calls helps make tests faster, stable, and predictable.
What are some common pitfalls when testing asynchronous code?
Some common pitfalls when testing asynchronous code include not waiting for the async operation to finish, forgetting to use async/await, not using proper waiting functions, incorrectly mocking APIs, and creating flaky tests that sometimes fail due to timing issues.
