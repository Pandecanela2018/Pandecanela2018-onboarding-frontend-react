What are the benefits of using React Testing Library instead of testing implementation details?W\
The benefits of testing all your components are that they verify the components work as expected and you do not need to verify them manually.
What challenges did you encounter when simulating user interaction?
The actions a tester performs follow predefined steps to verify that the system works. However, a user does not follow predefined steps and may find errors that the testers did not notice.
